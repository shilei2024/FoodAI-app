# 云函数目录

此目录包含可选的云函数代码，用于生产环境部署。

## 使用说明

### 当前项目模式
当前项目主要使用 **直接调用模式**：
- AI服务：直接调用Deepseek API
- 数据存储：本地存储
- 支付功能：需要企业认证和云函数

### 云函数用途
1. **baidu-ai/** - 百度AI代理（备选方案）
2. **create-order/** - 创建支付订单
3. **payment-callback/** - 支付回调处理

### 何时需要使用云函数？
- 生产环境保护API密钥
- 需要支付功能（企业认证）
- 需要数据云端同步

### 快速开始（直接调用模式）
如果您只想快速体验核心功能：
1. 配置Deepseek API密钥
2. 无需部署云函数
3. 直接使用AI识别和营养分析

详细配置请参考根目录的 `CONFIGURATION.md`。
