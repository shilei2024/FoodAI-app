# AI轻食记 - 用户状态管理功能演示

## 功能亮点

### 🎯 核心功能
1. **智能用户状态检测** - 自动识别游客/登录用户
2. **精细化权限控制** - 不同状态不同权限
3. **使用次数限制** - 保护服务资源，提升用户体验
4. **直观的UI提示** - 清晰的引导和反馈

### 👤 用户状态管理
- **游客状态**：所有功能受限，引导登录
- **登录状态**：基础功能可用，每日次数限制
- **VIP状态**：预留接口，可扩展无限使用

### 📊 使用次数限制
- **拍照识别**：每天5次（登录用户）
- **搜索识别**：每天10次（登录用户）
- **实时显示**：剩余次数一目了然
- **智能提示**：次数用完时友好提示

## 使用流程

### 1. 首次使用（游客状态）
```
首页显示：
┌─────────────────────────┐
│ 👤 游客                 │
│ 📸 拍照识别：0/5次      │
│ 🔍 搜索识别：0/10次     │
│ ⚠️ 请登录后使用完整功能 │
└─────────────────────────┘

功能限制：
- 拍照按钮 → 弹出登录提示
- 搜索功能 → 弹出登录提示
- 所有AI功能 → 无法使用
```

### 2. 用户登录
```
点击"登录"按钮：
┌─────────────────────────┐
│ ✅ 登录成功！           │
│ 👤 用户1234             │
│ 📸 拍照识别：0/5次      │
│ 🔍 搜索识别：0/10次     │
└─────────────────────────┘

功能解锁：
- 拍照识别 ✓（每日5次）
- 搜索识别 ✓（每日10次）
- 历史记录 ✓
- 个人设置 ✓
```

### 3. 使用功能
```
使用拍照识别：
第1次：剩余4次
第2次：剩余3次
...
第5次：剩余0次
第6次：弹出"今日拍照识别次数已用完"

使用搜索识别：
第1次：剩余9次
第2次：剩余8次
...
第10次：剩余0次
第11次：弹出"今日搜索识别次数已用完"
```

### 4. 个人页面
```
"我的"页面显示：
┌─────────────────────────┐

│ 📊 数据统计             │

包含：
```
